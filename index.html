Traveler’s Safety Web App

Technologies: HTML, CSS, JavaScript, Python, Machine Learning, MySQL

The Traveler’s Safety Web App is designed to ensure safe travel experiences for women by providing real-time location tracking and instant emergency alerts. The system verifies drivers using RTO data and employs OCR-based vehicle number recognition for enhanced authenticity. It enables users to share their live location with trusted contacts and send distress signals in one click. The web application integrates map APIs for route visualization and backend data storage for monitoring and reporting. This project emphasizes security, accessibility, and user-centric design.

Tools & Technologies

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Python / Flask
  • Database: MySQL
  • APIs: Google Maps API (for live tracking)
  • Machine Learning: OCR (Optical Character Recognition) for number plate detection
  • Other: Twilio / SMTP (for emergency alerts via SMS or email)

Food Recipe Data Collector

Technologies: PHP, HTML, CSS, SQL

The Food Recipe Data Collector is a web-based application developed to efficiently manage and store recipe information in a structured database. This project helped in understanding the integration of frontend design with backend data operations.
